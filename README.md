# Nic Chat Bot

## Descrição
Este é um chatbot simples em Python que responde perguntas sobre aprender Python. Ele pede o nome e email do usuário, oferece opções de perguntas e fornece respostas formais sobre tópicos como carreira, tempo de aprendizado e recomendações de cursos.

## Funcionalidades
- Saudação inicial e coleta de dados básicos (nome e email).
- Menu interativo com 5 opções:
  - [1] Vale a pena aprender Python?
  - [2] Quanto tempo leva para conseguir um emprego?
  - [3] Quando saber se estou bom o suficiente?
  - [4] Recomendações de onde estudar.
  - [5] Sair.
- Respostas personalizadas com o nome do usuário.
- Loop contínuo até o usuário escolher sair.

## Pré-requisitos
- Python 3.x instalado (baixe em [python.org](https://www.python.org)).
- Nenhum módulo externo necessário (usa apenas `os`, que vem com Python).

## Como Instalar e Executar
1. Clone este repositório: `git clone https://github.com/seu-usuario/nome-do-repo.git`
2. Navegue até a pasta: `cd nome-do-repo`
3. Execute o script: `python chatbot.py` (ou `python3 chatbot.py`, dependendo do seu sistema).

## Como Usar
- Execute o programa.
- Digite seu nome e email quando solicitado.
- Escolha uma opção do menu digitando 1, 2, 3, 4 ou 5.
- Para sair, selecione 5.
